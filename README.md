# luna-base

Lua / TypeScriptで記述可能なグラフィクスエンジン。

## 特徴
 * LuaまたはTypeScriptで記述できる
 * Luaデバッガを使用可能
 * ソースコードの大部分をTypeScriptで記述
 * WASMとWindowsネイティブで動作
